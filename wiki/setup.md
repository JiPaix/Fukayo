# Setup

## Install dependencies
```cmd
npm install
```

## Start Fukayo
### Hot module reload
The application is reloaded whenever changes occures.
```cmd
npm run watch
```
### Build and start
Build all packages and run electron.
```cmd
npm run build
npm run start
```
### Compile
Creates an executable.  
Files are located in the `$PROJECT_ROOT/dist` folder
```cmd
run npm compile
```
